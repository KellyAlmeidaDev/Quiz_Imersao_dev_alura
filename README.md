# Quiz Interativo Simples - Imersão Dev Alura
Este repositório contém o projeto de um Quiz Simples de Perguntas e Respostas, desenvolvido como parte dos exercícios da Imersão Dev Front-end da Alura. 
O projeto é uma ferramenta de aprendizado focada na aplicação prática dos fundamentos do desenvolvimento web, estruturado para ser uma base sólida para a construção de interfaces interativas.
 
## Objetivo de Aprendizagem
O principal objetivo deste projeto é fixar e consolidar os comandos principais e a sintaxe fundamental de:
* JavaScript: O foco é na lógica de manipulação de dados, estruturas condicionais para checar as respostas, e controle de fluxo para avançar pelas perguntas;
* CSS: Prática na estilização de elementos (cores, layout, responsividade básica) para apresentar as perguntas de forma clara;
* HTML: Estrutura e marcação dos elementos do quiz (perguntas, opções de resposta e área de pontuação).

## Tecnologias Utilizadas
O Quiz é construído usando a tríade fundamental do desenvolvimento front-end:
* **HTML**: Cria a estrutura estática do quiz, onde o JavaScript irá inserir o conteúdo dinamicamente;
* **CSS**: Responsável por tornar o quiz visualmente agradável e funcional em diferentes telas;
* **JavaScript**: Parte principa do projeto. É aqui que implementamos: armazenar o conjunto de perguntas, opções e a resposta correta, funções para carregar a próxima pergunta e verificar a resposta do usuário e atualização do conteúdo na tela em tempo real (exibição de perguntas, feedback de acerto/erro, e pontuação final).

## Como Rodar o Projeto
* Clone este repositório para sua máquina.
* Abra o arquivo index.html no seu navegador.
* Comece a responder as perguntas e teste a lógica por trás do quiz!
